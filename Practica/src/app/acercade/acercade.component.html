Acerca de esta aplicación

Esta aplicación web ha sido desarrollada con Angular y Bootstrap.

Permite buscar información sobre ciudades de todo el mundo utilizando la GeoDB Cities API.

Proyecto desarrollado como parte de un ejercicio práctico para el módulo de Desarrollo Web Entorno Cliente.

Tecnologías utilizadas:
  • Angular
  • Bootstrap 5
  • TypeScript
  • GeoDB API (via RapidAPI)
